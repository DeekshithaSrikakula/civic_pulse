# AGENTS

## Overview

CivicPulse uses AI-assisted processing to analyze civic complaints and generate actionable outputs.

## Complaint Analysis Agent

Responsibilities:
- Read complaint text
- Identify issue category
- Extract key information

Input:
- Telugu complaint

Output:
- Classified issue category

## Department Mapping Agent

Responsibilities:
- Match complaint category to the responsible department

Output:
- Department recommendation

## Complaint Summary Agent

Responsibilities:
- Generate structured complaint summaries
- Standardize complaint format

Output:
- Complaint summary ready for submission

## Supported Categories

- Garbage Management
- Water Supply
- Drainage
- Roads and Potholes
- Street Lighting
- Public Sanitation
