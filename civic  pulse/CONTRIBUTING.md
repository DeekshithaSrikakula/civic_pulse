# Contributing

Thank you for contributing to CivicPulse.

## How to Contribute

1. Fork the repository.
2. Create a new branch.
3. Implement your changes.
4. Test the application.
5. Submit a pull request.

## Guidelines

* Follow Python coding standards.
* Keep code modular and readable.
* Add comments where required.
* Update documentation when necessary.

## Bug Reports

Please include:

* Issue description
* Steps to reproduce
* Expected behavior
* Actual behavior

## Feature Requests

Clearly describe:

* Proposed feature
* Problem it solves
* Expected outcome
