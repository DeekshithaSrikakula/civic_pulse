# User Manual

## Introduction

CivicPulse helps analyze civic complaints written in Telugu and provides structured outputs for efficient grievance handling.

## Steps to Use

### Step 1

Launch the application.

```bash
streamlit run app.py
```

### Step 2

Enter a civic complaint in Telugu.

Example:

మా కాలనీలో మూడు రోజులుగా చెత్త తీసుకెళ్లడం లేదు.

### Step 3

Click Submit.

### Step 4

View results:

* Complaint Category
* Responsible Department
* Complaint Summary

## Supported Categories

* Garbage Issues
* Water Supply Issues
* Drainage Problems
* Road Damage
* Street Light Issues
* Public Sanitation Issues

## Expected Output

Category: Garbage

Department: GHMC Sanitation Wing

Summary: Garbage has not been collected in the locality for three days.
